# Discord Clone 🦊🐾

Красивый и функциональный клон Discord на React 19, Vite и Zustand. Приложение воссоздает панели навигации Discord, личные сообщения, список друзей, голосовые звонки с графиками активности в реальном времени и форму редактирования профиля с валидацией.

---

## 🚀 Инструкция по запуску

После клонирования репозитория выполните следующие шаги:

1. **Установка зависимостей**
   Поскольку для построения графиков используется библиотека `recharts`, а проект работает на новейшем React 19, установите зависимости с флагом `--legacy-peer-deps`:
   ```bash
   npm install --legacy-peer-deps
   ```

2. **Запуск сервера для разработки**
   Запустите сервер Vite в режиме разработки:
   ```bash
   npm run dev
   ```
   Откройте ссылку в браузере (обычно [http://localhost:5173](http://localhost:5173)).

3. **Сборка проекта**
   Скомпилируйте проект для продакшена:
   ```bash
   npm run build
   ```

## 🛠️ Стек технологий и библиотеки

Проект разработан с использованием следующих ключевых библиотек:
* **React 19** & **Vite** — быстрая сборка и высокая производительность.
* **React Router Dom** — организация роутинга и переходов по страницам (синхронизация URL со стором).
* **Zustand** — легкое и масштабируемое управление состоянием приложения с авто-сохранением в `localStorage`.
* **React Hook Form** + **Zod** — валидация форм настроек по схемам с выводом ошибок.
* **Recharts** — интерактивные графики для отображения задержки RTC-соединения и громкости голоса.
* **Lucide React** — красивый набор современных иконок.

## 🌟 Основной функционал

1. **Discord Layout & Navigation**:
   * Левая колонка со значками серверов (включая главную кнопку в стиле Discord).
   * Боковая панель каналов (текстовые каналы и голосовые) или списка ЛС (при нахождении на Главной).
   * Правая панель участников сервера (показывает текущего юзера и список друзей в зависимости от статуса).
   * Нижний блок профиля пользователя со статусом, ником и кнопками быстрого мьюта звука / открытия настроек.

2. **Страница Друзей (Главный экран)**:
   * Фильтрация друзей по вкладкам: *В сети*, *Все*, *Ожидание*, *Заблокированные*.
   * Поиск друзей в реальном времени.
   * Сортировка списка друзей по алфавиту или по статусу активности.
   * Добавление новых друзей через форму с мгновенным визуальным фидбеком и автоматическим подтверждением запроса через 5 сек (имитация активности).

3. **Текстовые Чаты (Серверные каналы & Личные Сообщения)**:
   * Загрузка истории сообщений из Zustand.
   * Отправка сообщений и автоматическая прокрутка чата вниз.
   * Поиск по сообщениям в текущем канале.
   * **Автоматические ответы (чат-бот)**: при отправке слов "привет", "hello" или "femboy", бот напишет забавный и милый ответ через 1.5 секунды с анимацией печати.

4. **Голосовой Звонок (RTC Simulation)**:
   * Подключение при клике на голосовые каналы серверов.
   * Карточки участников с анимированными кругами вокруг аватаров, реагирующими на звук (Voice Activity).
   * Панель управления звонком: мьют микрофона, выключение звука (оглушение), включение камеры, стрим экрана, кнопка отключения звонка.
   * **Realtime-графики (Recharts)**:
     * *Задержка сети (Ping)* — обновляемый в реальном времени график пинга (мс). При включении стрима экрана пинг реалистично повышается.
     * *Уровень активности звука* — динамический столбчатый график громкости каждого участника звонка.

5. **Модальное окно настроек профиля**:
   * Форма редактирования юзернейма, кастомного статуса, цвета аватара и темы карточки.
   * Валидация на базе **Zod**: юзернейм должен быть от 3 до 15 символов и состоять только из латинских букв, цифр и `_`. Кастомный статус — не более 50 символов.
   * Интерактивный предпросмотр (Live Preview) карточки профиля прямо в модальном окне при вводе данных.
